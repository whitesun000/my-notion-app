# My Personal Notion (Python Edition)

PythonとStreamlitで作成した、自分専用のNotion風メモアプリです。

## 特徴
- **MVCモデル**に基づいたクリーンな設計
- **SQLite**によるローカルデータ保存
- **オブジェクト指向**（継承・カプセル化）を活用したブロック管理

## 使い方
1. ライブラリのインストール
   `pip install -r requirements.txt`
2. アプリの起動
   `streamlit run main.py`
